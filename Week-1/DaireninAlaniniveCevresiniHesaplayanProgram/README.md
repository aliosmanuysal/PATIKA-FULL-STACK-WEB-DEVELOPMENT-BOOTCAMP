# Dairenin alanını ve hesaplayan program.
* Kullanıcıdan alınan yarıçapı kullanarak dairenin alanını ve çevresini hesaplayan program.
#### Formül:
```plaintext
    Alan = π * r * r
    Çevre = 2 * π * r
    π = 3.14
```
#### Çıktı:
```plaintext
    Yarıçap: 5
    Alan: 78.5
    Çevre: 31.4
```
