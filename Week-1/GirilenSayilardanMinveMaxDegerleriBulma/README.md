# Girilen Sayılardan Min ve Max Değer Bulma
* Kullanıcıdan alınan n tane sayı arasından en büyük ve en küçük sayıyı bulan program.
  
## Senaryo
```plaintext
Kaç tane sayı gireceksiniz: 4
1. Sayıyı giriniz: 16
2. Sayıyı giriniz: -22
3. Sayıyı giriniz: -15
4. Sayıyı giriniz: 100
En büyük sayı: 100
En küçük sayı: -22
```
